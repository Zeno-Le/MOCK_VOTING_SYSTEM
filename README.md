# Mock Voting System

A simple command-line program that simulates a basic election between two candidates. This project was created as a learning exercise to practice Python fundamentals like loops, lists, user input, and randomization.

---

## Features

* Menu-driven interface
* Option to enter custom candidates
* Option to use a predefined list of historical candidates
* Randomized vote generation and winner selection

---

## Getting Started

### Requirements

* Python 3.13

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Zeno-Le/MOCK_VOTING_SYSTEM.git
```

2. Navigate into the project folder:

```bash
cd MOCK_VOTING_SYSTEM
```

3. Run the script:

```bash
script.py
```

4. Follow the on-screen menu to select candidates or exit.

---

## Notes

* The vote generation is randomized and purely for simulation.
* Currently supports only two candidates per election.
* Candidate lists may not reset between runs.
* The project is a learning exercise and could be refactored to be more Pythonic and efficient.

---

## Learning Goals

* Working with lists and loops
* Handling user input
